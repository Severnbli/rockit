## Description

Unity 2D puzzle-platformer about putting order in space.

### Used packages

* Unity 6000.3.7f1.
* [Odin](https://odininspector.com/) 4.0.1.4. `#proprietary`
* [LeoEcs Proto](https://gitverse.ru/leopotam/ecsproto) 2025.11.4 (extensions: [QoL](https://gitverse.ru/leopotam/ecsproto-qol) 2025.9.4-0, [Conditional Systems](https://gitverse.ru/leopotam/ecsproto.conditionalsystems) 2025.9.4-0, [Unity](https://gitverse.ru/leopotam/ecsproto.unity) 2025.11.4, [Unity uGui](https://gitverse.ru/leopotam/ecsproto.unity.ugui) 2025.9.4-0, [Unity Physics2D](https://gitverse.ru/leopotam/ecsproto.unity.physics2d) 2025.9.4-0).
* [Zenject](https://github.com/modesttree/Zenject) 9.2.0.
* [UniTask](https://github.com/Cysharp/UniTask) 2.5.10.
* [DOTween](https://dotween.demigiant.com/index.php) 1.2.825.
* [NuGetForUnity](https://github.com/GlitchEnzo/NuGetForUnity) 4.5.0.
* [Newtonsoft.Json](https://github.com/jamesnk/newtonsoft.json) 13.0.4.
